$database->scan (string $string);

$database->scan also adds any class whose full name (\path\to\class) begins on given $string

If we have:

\animals\wolf
\ant
\animation

Than

$database->scan ('.ani');

Or

$database->scan ('\\ani');

Or

$database->scan ('\ani');

Will add \animals\wolf and \animation classes table handlers to orm.
